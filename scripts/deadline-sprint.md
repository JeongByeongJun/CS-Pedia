# 데드라인 일괄 업데이트 스프린트

최신 연도 기준 다음 사이클 데드라인 발표 여부 조사.
조사 후 `deadlines.json` 업데이트 → `pnpm seed`

- ✅ = 이미 최신 or 완료
- 🔍 = 조사 필요
- ❌ = 미발표 / 없음
- `-` = 데드라인 데이터 없음 (신규 추가 대상)
