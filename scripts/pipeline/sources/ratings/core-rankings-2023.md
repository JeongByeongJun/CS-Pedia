## 참고

- CORE 2023은 가장 최신 공식 랭킹
- TBR인 DAC, MICRO, RSS는 실제로는 top-tier급이나 CORE가 평가 미완료

- Journal 분류된 학회는 논문이 저널에 게재되는 형식이라 학회 등급 대신 저널 등급 적용
